1. Giới thiệu MÕM Tool

MÕM Tool là phần mềm khôi phục mật khẩu file Office (Excel, Word, PowerPoint) chạy trực tiếp trên máy tính Windows, không cần cài đặt, không gửi file lên mạng.

Hỗ trợ định dạng: .xlsx, .xls, .xlsb, .xlsm, .docx, .doc, .pptx, .ppt

Phương pháp: Brute Force (vét cạn), Từ điển (Wordlist), Khoảng số, VBA Crack

💡 Tool chạy 100% offline — file của bạn không rời khỏi máy tính.

2. Tính năng cập nhật mới

✅ Hỗ trợ pass dài 10+ ký tự — không còn đơ

Phiên bản cũ bị đơ/crash khi đặt max length ≥ 10 vì cố load toàn bộ hàng tỷ candidates vào RAM. Bản mới dùng batch feeding: chia nhỏ 500K/đợt, workers xử lý xong đợt này mới lấy đợt tiếp.

✅ ETA hiển thị năm-tháng-ngày-giờ

Thay vì hiển thị số ngày khổng lồ khó đọc, giờ hiển thị dạng ~2y 3mo 15d 8h hoặc ~5d 14h tùy độ lớn.

✅ Cảnh báo khi candidates > 50 triệu

Popup cảnh báo kèm ước tính thời gian thực tế trước khi bắt đầu — giúp bạn quyết định có nên thu hẹp bộ ký tự không.

✅ GUI cập nhật ngay khi bắt đầu

Trước đây W1-W8 không hiển thị ngay — phải ấn tạm dừng rồi tiếp tục mới thấy. Đã sửa: GUI tự chờ và cập nhật ngay.

💡 Dù hỗ trợ pass 10+ ký tự, brute force toàn bộ charset 62 ký tự ở độ dài 10 cần ~52 năm. Dùng Pattern nếu biết một phần để giảm xuống vài giây.

4. Chọn file cần khôi phục

  1. Nhấn nút Browse hoặc kéo thả file vào ô chọn file
  2. Chọn file Excel/Word/PowerPoint cần khôi phục mật khẩu

💡 Tool tự nhận diện loại mã hóa: AES-256 (Office 2013+), AES-128 (Office 2007-2010), RC4 (Office 97-2003). File RC4 dò rất nhanh!

5. Chế độ Vét cạn (Brute Force)

Thử tất cả tổ hợp ký tự có thể. Phù hợp khi bạn biết mật khẩu gồm những loại ký tự nào.

Cách dùng:

  1. Chọn tab 🔍 Vét cạn
  2. Nhập bộ ký tự (hoặc click nút 0-9, a-z, A-Z, @)
  3. Đặt độ dài tối thiểu và tối đa
  4. Nhấn ▶️ Bắt đầu

Pattern (tùy chọn):

Nếu biết một phần mật khẩu, dùng * cho ký tự chưa biết. Ví dụ: abc* sẽ thử abc0, abc1, ..., abcz

💡 Ctrl+click các nút bộ ký tự để chọn nhiều loại cùng lúc.

6. Chế độ Từ điển

Thử các mật khẩu từ file danh sách. Nhanh hơn brute force nếu mật khẩu phổ biến.

  1. Chọn tab 📚 Từ điển & Số
  2. Nhấn 📂 Chọn File → chọn file .txt chứa danh sách mật khẩu
  3. Nhấn ▶️ Bắt đầu
💡 Tải wordlist phổ biến: rockyou.txt, common-passwords.txt từ GitHub. Mỗi dòng là 1 mật khẩu.

7. Chế độ Khoảng số

Thử tất cả số trong một khoảng. Phù hợp khi mật khẩu là số (ngày sinh, năm, mã số...).

  1. Chọn tab 📚 Từ điển & Số
  2. Nhập số bắt đầu và số kết thúc
  3. Hoặc dùng preset nhanh: 0000-9999, 000000-999999, 1990-2030

8. Gỡ khóa VBA Macro

Dành cho file .xlsm, .xlsb, .xls có VBA bị khóa (không xem/sửa được code).

  1. Chọn file Excel có VBA ở ô chọn file trên
  2. Chọn tab 🔧 VBA Crack
  3. Nhấn 🔍 QUÉT VBA TRONG FILE ĐÃ CHỌN
  4. Chọn cách xử lý:
    - Cách 1: XÓA PASS — đổi DPB thành DPx, mở Excel sẽ hỏi sửa lỗi → chọn Yes
    - Cách 2: ĐÈ PASS '123' — đặt mật khẩu VBA thành 123
⚠️ Cần mở lại file trong Excel sau khi xử lý. Nếu file bị mã hóa, cần tìm pass mở file trước.

9. Đa luồng X1-X8 & tạm dừng từng luồng

Chia toàn bộ candidates thành các chunk, mỗi luồng xử lý 1 chunk song song — giống như mở nhiều app cùng lúc nhưng trong 1 app duy nhất.

  • X1 — 1 luồng, dùng 1 nhân CPU. Phù hợp máy yếu, CPU 2 nhân, hoặc tài khoản FREE
  • X2 — 2 luồng, dùng 2 nhân CPU. Nhanh ~2x so với X1. Cần CPU ≥ 2 nhân
  • X3/X4/X5 — 3-5 luồng. Cần CPU ≥ 4 nhân, chỉ PRO
  • X6/X7/X8 — 6-8 luồng. Nhanh nhất. Cần CPU ≥ 6 nhân, chỉ PRO
💡 Kiểm tra số nhân CPU: Chuột phải vào Taskbar → Task Manager → tab Performance → CPU → xem "Cores". Chọn số luồng ≤ số nhân CPU thực tế.
💡 Mỗi luồng hiển thị riêng W1-W8 với candidate đang thử, tiến độ, timer riêng và ETA. Có thể tạm dừng/tiếp tục từng luồng riêng lẻ bằng nút ⏸/▶. Luồng nào tìm thấy sẽ hiện ✅ TÌM THẤY!
⚠️ Dùng X8 trên CPU 4 nhân sẽ không nhanh hơn X4, thậm chí chậm hơn do overhead. Chọn số luồng ≤ số nhân CPU thực tế.

10. Mẹo tăng tốc độ

  • Biết mật khẩu có số → chỉ dùng bộ ký tự 0-9
  • Biết độ dài → đặt min = max = độ dài đó
  • Biết một phần → dùng Pattern với dấu *
  • File .xls cũ (RC4) → dò nhanh hơn nhiều so với .xlsx (AES-256)
  • Dùng X4 trên máy có CPU mạnh (4+ core)
  • Tắt các ứng dụng khác để dành CPU cho tool
